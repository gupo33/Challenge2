Detailed Description

template<typename T, StorageOrder stor>
class algebra::Matrix< T, stor >

Template Matrix class encoded in COOmap form, and compressible in CSR / CSC form, depending on the ordering.

Template Parameters
Ttype of the elements of the Matrix
storIndicates how the Matrix is ordered (row-column major)
